# 🏡 Airbnb Listing Analysis – Pricing & Demand Insights

## 📌 Project Overview
This project analyzes Airbnb listing data to identify key factors affecting pricing, availability, and customer demand. The objective is to derive actionable insights that help hosts optimize their listings and maximize revenue.

## 🎯 Business Problem
Airbnb hosts often face challenges such as:
* How to price listings competitively
* Which locations generate higher revenue
* What factors influence booking frequency

This project answers:
* Which neighbourhoods have the highest prices?
* How room type impacts pricing and demand
* How reviews and availability affect bookings

## 📊 Dataset Description
The dataset includes Airbnb listings with the following features:
* **Neighbourhood Group & Area** – Location of listing
* **Room Type** – Entire home, private room, shared room
* **Price** – Cost per night
* **Number of Reviews** – Customer engagement indicator
* **Availability (365 days)** – Listing availability per year

## 🛠️ Tools & Technologies Used
* **Excel** – Pivot Tables and Charts
* **Python (Pandas, Matplotlib)** – Data cleaning & EDA

## 🔄 Project Workflow
1. Cleaned dataset (handled missing values, removed duplicates)
2. Performed exploratory data analysis (EDA)
3. Made Pivot tables to extract key business insights
4. Interpreted results to generate business recommendations

## 📈 Key Insights
* **Entire homes/apartments** have the highest average prices compared to other room types
* **Manhattan/Brooklyn** dominates in total listings and revenue potential
* Listings with **higher reviews tend to have better occupancy rates**
* **Private rooms are more affordable**, attracting budget-conscious customers
* Listings with **limited availability often have higher pricing**, indicating high demand

## 💡 Business Recommendations
* Hosts should prioritize listings in **high-demand neighbourhoods** for better returns
* Improve customer experience to **increase reviews and visibility**
* Use **dynamic pricing strategies** based on demand and availability
* Offer competitive pricing for private rooms to boost occupancy rates
