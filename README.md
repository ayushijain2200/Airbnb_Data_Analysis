# Airbnb_Data_Analysis
Performed EDA on Airbnb dataset to enhance guest experience and improve host performance
In this project, I performed Exploratory Data Analysis (EDA) on an Airbnb dataset to understand pricing patterns, room type distribution, geographical trends, and customer engagement through reviews. 
I used Python libraries like Pandas, Matplotlib, and Seaborn for data cleaning and visualization.
I also used MS Excel to clean the dataset and created pivot tables and charts to perform EDA.
